Fuld Video

Samlet set fremstiller vores kampagnefilm nogle karikerede scenarier, der tydeligt fremhæver, hvor mange muligheder og fordele der er ved at vælge Tuborg NUL. Dette gør sig særligt gældende for folk, der minder om vores personaer, da de ønsker at nå så meget som muligt med deres tid. Derfor står Tuborg NUL som det perfekte valg, efter at man har set vores film.
